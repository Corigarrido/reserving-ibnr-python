# Reserving IBNR in Python

## Project Goal

Develop a Python application that implements the Chain Ladder method to estimate IBNR reserves from claims development triangles.

---

## Business Problem

Insurance companies often estimate IBNR reserves using spreadsheets, which may be difficult to maintain, validate and automate.

The objective of this project is to build a reproducible and transparent workflow using Python.

---

## Inputs

- Claims data (.xlsx)
- Development periods
- Accident periods

---

## Outputs

- Development Triangle
- Development Factors
- Cumulative Development Factors
- Ultimate Losses
- IBNR Reserve
- Charts
- Excel Report

---
